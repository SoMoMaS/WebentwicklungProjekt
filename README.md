# WebentwicklungProjekt

To start the project do the followings in the given order:

- install all dependencies

```sh
$ npm install -d
```

- start the rethinkDB

```sh
$ ./db/rethinkdb.exe
```

- start the backend
```sh
$ cd ./backend/src
$ node server.js
```

- start the frontend
```sh
$ cd ./frontend/app
$ ng serve
```
